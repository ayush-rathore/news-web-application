## Run Locally

Clone the project

```bash
  git clone https://github.com/ayush-rathore/news-web-application/
```

Go to the project directory

```bash
  cd news-web-application
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
